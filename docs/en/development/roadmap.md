# Roadmap


This document tracks Prismctl's design principles and release plan.

## Current status

- v0.1.0: completed (local tag: `v0.1.0`)

## v0.2.0 (planned)

- Status overview commands (e.g. `prismctl status`)
- Backup/restore flows
- Template listing and diffing
- Install skills from URL/GitHub

## v0.3.0 (planned)

- Distribution improvements (e.g. Homebrew tap)
- CLI refactor (possibly clap)
- Error handling standardization (e.g. thiserror)
