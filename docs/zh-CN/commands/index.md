# 命令参考


本节是 Ekko CLI 的命令参考，按功能拆分为多个页面。

## 全局选项

| 选项 | 说明 |
|------|------|
| `--home <PATH>` | 重定向所有读写到指定目录（HOME 沙箱） |
| `--dry-run` | 仅预览变更，不实际写入（默认） |
| `--apply` | 实际执行变更 |
| `--lang <LANG>` | 模板语言：`zh-CN`（默认）或 `en` |
| `-h, --help` | 显示帮助信息 |
