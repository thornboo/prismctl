# Codebase Visualizer

Generate an interactive HTML tree view showing project file structure.

## Usage

Run from project root:
```bash
python ~/.claude/skills/codebase-visualizer/scripts/visualize.py .
```

Creates `codebase-map.html` and opens in browser.
